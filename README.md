# 📊 Sales Dashboard

## 📌 Project Overview

This interactive Power BI dashboard provides comprehensive insights into sales performance, revenue, profit, regional trends, customer segments, and product analysis. The dashboard enables business users to make data-driven decisions through interactive filtering and KPI monitoring.

---

## 🎯 Business Objectives

- Monitor total sales and profit
- Track monthly sales performance
- Identify top-performing products
- Analyze customer segments
- Compare regional performance
- Support strategic business decisions

---

## 🛠️ Tools Used

- Power BI
- Microsoft Excel
- Power Query
- DAX
- Data Modeling

---

## 📈 Dashboard Features

- Revenue KPI
- Profit KPI
- Sales Trend Analysis
- Product Performance
- Regional Analysis
- Customer Segment Analysis
- Interactive Filters
- Dynamic Visualizations

---

## 📊 Key Insights

- Monthly sales trends
- Top-performing products
- Regional revenue comparison
- Customer segment contribution
- Profitability analysis
